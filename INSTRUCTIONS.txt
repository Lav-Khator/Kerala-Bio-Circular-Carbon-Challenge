## üõ†Ô∏è Manual Setup & Running

If the batch file doesn't work or you are on Mac/Linux:

### 1. Install Python
Ensure you have Python installed. You can download it from [python.org](https://www.python.org/downloads/).

### 2. Install Dependencies
Open a terminal/command prompt in this folder and run:
pip install -r requirements.txt

### 3. Run the Dashboard
In the terminal, run:
python -m http.server 8000

Then open your browser to:
http://localhost:8000/dashboard_website/dashboard_beautiful.html

### 4. Re-running the Solver (Optional)
If you want to re-run the simulation logic:
python algorithm_code/run.py

This will update `solution/solution.csv`.
